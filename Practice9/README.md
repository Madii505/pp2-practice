# Practice 09: Game Development with Pygame

Коллекция из трех игр на Pygame:

1. **Mickey's Clock** - цифровые часы с руками Микки Мауса
2. **Music Player** - плеер с клавиатурным управлением
3. **Moving Ball** - игра с движущимся мячом

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
# Mickey's Clock
python mickeys_clock/main.py

# Music Player
python music_player/main.py

# Moving Ball
python moving_ball/main.py
```

## Управление

### Mickey's Clock
- Автоматически показывает текущее время
- Обновляется каждую секунду

### Music Player
- **P** - Воспроизведение
- **S** - Остановка
- **N** - Следующий трек
- **B** - Предыдущий трек
- **Q** - Выход

### Moving Ball
- **↑** - Вверх
- **↓** - Вниз
- **←** - Влево
- **→** - Вправо
