# Moving Ball Game

Интерактивная игра с движущимся мячом

## Правила
- Красный мяч (50x50 пикселей)
- Управление стрелками клавиатуры
- Мяч не может выходить за границы экрана
- Каждое нажатие = движение на 20 пикселей

## Управление
- **↑** - Вверх
- **↓** - Вниз
- **←** - Влево
- **→** - Вправо
- **Q** - Выход
